# Duku Ltd — steel fixing mockup

Prospect mockup. Steel fixing contractor, Southeast England. No prior logo or web presence,
so the logo was built from scratch (a rebar "D" monogram) and the photos are free-licence
placeholders until Duku supply their own job photos.

**Live:** https://paddydon-a11y.github.io/mockups-site/duku-steel-fixing/

## Recipe
`archetype:A-light / hero:H2 / services:S2 / gallery:G1 / reviews:R1 / trust:T1 / contact:C1 / flourish:big-shoulders-display`

Bright cool-stone surface, charcoal text, safety-orange accent (`#FF6A1A`) derived from the
logo. Big Shoulders Display headlines (industrial), Inter body.
