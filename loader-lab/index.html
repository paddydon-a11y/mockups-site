Trade Loader Lab

Internal reference for trade-themed loading screens. Pick the concept that matches the prospect's trade, lift the code into the mockup, rebrand the colours and wordmark. The loader counts as the mockup's distinctive flourish. Full rules in construction-sites/MOCKUP-SYSTEM.md → "Trade loaders". Click any card to play it full-screen.

Hard rules

  • 2.5 seconds max, then a fail-safe timeout removes the loader even if assets are still pending.
  • Page content renders underneath — never gate content on JS.
  • transform/opacity animations only. No layout thrash, smooth at 390px.
  • prefers-reduced-motion: skip the theatrics, show the brand card, fade fast.
  • Mockups: play on every load (it's the pitch). Live client sites: gate behind sessionStorage so it plays once per session.
  • One loader concept per mockup, executed cleanly. Check recipe-ledger.json — no concept repeats within the last 10 mockups.
